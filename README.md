# ボートレース予測システム マニュアル（ドラフト）

このディレクトリは、ボートレース予測AIシステムのユーザー向け・管理者向け操作マニュアルを整理するためのドラフトです。最終的には GitHub Pages での公開および PDF 出力を想定しています。

## ディレクトリ構成

```
boatrace-manual/
├── index.html        # ポータルページ（ユーザー/管理者マニュアルへのリンク）
├── styles.css        # 共通スタイルシート
├── user/
│   └── index.html    # ユーザー向けマニュアル本体
├── admin/
│   └── index.html    # 管理者向けマニュアル本体
├── assets/
│   └── images/       # 画面キャプチャなどの画像を格納
└── README.md         # このファイル
```

## 収録内容

- **ユーザー向けマニュアル**: ログイン、最新予測ダッシュボード、予測履歴、アカウント設定、FAQ。操作ステップごとのスクリーンショット差し替え用プレースホルダーあり。
- **管理者向けマニュアル**: アカウント発行・一括登録・パスワードリセット・障害対応・FAQ。各操作フローを細分化し、スクリーンショット配置の目安を示しています。
- 各マニュアルにはスクリーンショット挿入用のプレースホルダーを用意しています。

## スクリーンショットの追加方法

1. `assets/images/` に PNG / JPG などの画像ファイルを保存します。
2. `user/index.html` や `admin/index.html` 内の `<img src="../assets/images/xxx.png">` のファイル名を実際の画像に合わせて更新します。
3. 必要に応じて `<figcaption>` や説明文も調整してください。

## PDF 出力（案）

- ブラウザで対象の HTML を開き、印刷ダイアログ（`⌘ + P`）から PDF として保存する。
- もしくは `wkhtmltopdf` や `pandoc` を利用してコマンドラインから変換する。
